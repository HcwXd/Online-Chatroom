# Online-Chatroom

> Slack-like Online chatroom built with Node.js, Express, Socket.io, MongoDB, mLab 



## Installation

```
$ git clone https://github.com/HcwXd/Online-Chatroom.git
$ cd Online-Chatroom
$ npm install 
```



## Usage

```
$ npm run start

# demo page: http://localhost:3000/
```
